# LLM Crash Course - Gen AI App

A comprehensive study application designed to help users accelerate understanding and applications for LLMs and Generative AI. This app enables users to track progress through LLM/AI topics, take rich text notes, and then generate custom study questions using Anthropic's Claude Sonnet-4.5 API. 10 modules (LLM Architecture Prompt Engineering, Model Optimization, Fine Tuning, GPU Acceleration and Optimization, etc), with 80 total readings.

## Features

- **Topic Tracking**: Organized view of all relevant topics and subtopics with progress indicators.
- **Rich Text Notes**: Integrated rich text editor for taking detailed notes on each reading and subtopic.
- **Progress Monitoring**: Visual progress bars and completion status for topics and readings.

- **Study Aids**:
  - Generate study questions and quizzes (AI-powered).
  - View summaries and study guides.
- **Responsive Design**: Optimized for both desktop and mobile viewing.

## Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **UI Library**: [React 18](https://react.dev/)
- **Styling**: Standard CSS with CSS Modules/Variables (no heavy UI frameworks)
- **Deployment & Storage**: [Vercel](https://vercel.com) (Analytics, Blob Storage)
